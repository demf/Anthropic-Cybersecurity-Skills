# Cybersecurity Skills for AI Agents

734 cybersecurity skills organized in **28 installable categories**. Pick only what you need.

> Fork of [mukul975/Anthropic-Cybersecurity-Skills](https://github.com/mukul975/Anthropic-Cybersecurity-Skills), restructured for selective category installation.

## Quick Start

```
/plugin marketplace add demf/Anthropic-Cybersecurity-Skills
```

Then select which categories to install from the plugin list.
